# Modern Nano Blue Portfolio Website

## Overview
A modern, responsive portfolio website with nano blue color theme. Features smooth animations, mobile menu, scroll effects, and fade-in animations.

## Quick Start
1. Open `index.html` in your browser:
   ```
   start index.html
   ```
   (On Windows) or drag to browser.

2. Customize:
   - Replace "YourName", email, links, projects with your info.
   - Add real project images to `images/` or update Unsplash URLs.
   - Edit skills/projects in `index.html`.

## Features
- **Responsive Design**: Mobile-first, works on all devices.
- **Nano Blue Theme**: Custom CSS variables for easy color changes.
- **Smooth Interactions**: Navbar scroll, hover effects, form submission.
- **Animations**: Intersection Observer for scroll-triggered fades.
- **Fonts/Icons**: Google Fonts (Inter), Font Awesome.

## File Structure
```
portfolio/
├── index.html      # Main page
├── css/
│   └── styles.css  # Modern styles + nano blue
├── js/
│   └── script.js   # Interactivity
└── README.md       # This file
```

## Customization Tips
- **Colors**: Edit `:root` variables in `styles.css`.
- **Projects**: Add more `.project-card` divs.
- **Contact**: Connect form to backend (e.g., EmailJS, Netlify Forms).

No build tools needed - pure HTML/CSS/JS!

Enjoy your portfolio 🚀
